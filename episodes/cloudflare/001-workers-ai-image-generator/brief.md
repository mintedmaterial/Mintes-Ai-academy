# Episode 001: Build an AI Image Lab with Cloudflare Workers AI

## Audience

Beginning-to-intermediate AI builders who want to understand Cloudflare and reproduce a real example.

## Viewer promise

By the end, the viewer understands the Workers AI binding and has the exact structure needed to deploy a Worker that accepts a prompt and returns an AI-generated image.

## Product demo

**Minte Image Lab**

A polished local-first interface with prompt presets, model and size controls, local history, download, request status, and a teaching panel.

## Target length

8-10 minutes.

## Visual language

Black, white, and Cloudflare orange. Editorial technical presentation. Real terminal output and real local UI. Slides explain the architecture, not decorate it.

## Safety boundary

The example is public and reproducible. The running inference demo is local or temporary. It must be stopped or deleted after recording.
