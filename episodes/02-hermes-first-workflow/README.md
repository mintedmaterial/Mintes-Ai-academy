# Episode 2 — Your First Useful Hermes Workflow

A Minte AI Academy tutorial by Cleo. This episode takes a new Hermes user from a clean install to a verified first workflow: setup, one real chat, a skill, a profile, and a scheduled task.

## Viewer promise
By the end, viewers can install Hermes, verify the provider before adding complexity, create an isolated profile, load a reusable skill, and schedule a safe recurring task.
