# Minte AI Academy

Public, public-safe teaching repository for practical Cloudflare and AI tutorials.

## Operating rules

- Never commit API tokens, account IDs, private URLs, dashboard screenshots, or internal infrastructure details.
- Demos are local or temporary. Do not leave unauthenticated inference endpoints deployed.
- Verify current Cloudflare documentation and model availability before recording.
- Keep examples small enough for a viewer to reproduce.
- Keep visible copy free of em-dashes.

## Commands

```bash
npm install
npm run dev
npm run build
npm run typecheck
npm run validate
```
